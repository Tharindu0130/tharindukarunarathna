## 💡 Pro Tips

### For Profile Pictures:
- Use a professional headshot or friendly casual photo
- Make sure your face is clearly visible
- Good lighting is essential
- Square crops work best for circular frames
- Tools: Canva, Photoshop, or remove.bg for backgrounds

- Use descriptive project names
- Keep your GitHub profile up to date

## 🆘 Troubleshooting

### Profile picture not showing?
1. Check file path: `public/images/profile.jpg`
2. Verify file extension (.jpg vs .jpeg)
3. Clear browser cache (Ctrl+Shift+Delete)
4. Check browser console for errors (F12)

### GitHub links not working?
1. Check internet connection
2. Verify GitHub username is correct
3. Try opening in incognito mode
4. Clear browser cache

### Layout looks weird?
1. Hard refresh browser (Ctrl+Shift+R)
2. Clear cache
3. Check if dev server is running
4. Restart dev server if needed
